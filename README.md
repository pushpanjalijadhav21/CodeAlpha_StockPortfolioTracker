# Stock Portfolio Tracker

## Description
This is a simple Python project that calculates the total investment of a stock portfolio using predefined stock prices. The user enters stock names and quantities, and the program displays the total investment.

## Features
- Uses a predefined dictionary of stock prices.
- Accepts stock name and quantity as input.
- Calculates the total investment.
- Displays a portfolio summary.
- Handles invalid stock names.

## Technologies Used
- Python

## How to Run
1. Clone or download this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the Python file:
   bash
   python CodeAlpha_stock port foliotracker.py
